# Структура проекта Planica

## Обзор

Приложение было реорганизовано с монолитного компонента в модульную архитектуру с разделением на компоненты, страницы, хуки и утилиты.

## Структура папок

```
src/
├── components/          # Переиспользуемые компоненты
│   ├── Header/         # Шапка приложения
│   ├── Sidebar/        # Боковое меню навигации
│   ├── UserMenu/       # Меню пользователя
│   ├── Notifications/  # Компонент уведомлений
│   ├── SearchBar/      # Поисковая строка
│   ├── Layout/         # Основной layout с роутингом
│   └── index.ts        # Экспорты компонентов
│
├── pages/              # Страницы приложения
│   ├── Home/          # Главная страница
│   ├── Sales/         # Страница продаж
│   ├── PageTemplate/  # Шаблон для страниц в разработке
│   └── index.ts       # Экспорты страниц
│
├── hooks/             # Кастомные React хуки
│   ├── useClickOutside.ts  # Хук для закрытия по клику вне элемента
│   ├── useCurrentTime.ts   # Хук для отображения текущего времени
│   └── index.ts       # Экспорты хуков
│
├── types/             # TypeScript типы и интерфейсы
│   └── index.ts       # Все типы приложения
│
├── utils/             # Утилиты и вспомогательные функции
│   └── navigation.ts  # Конфигурация навигации
│
├── App.tsx            # Главный компонент с роутингом
├── App.css            # Глобальные стили
└── index.tsx          # Точка входа
```

## Роутинг

Приложение использует React Router v6. Все маршруты определены в `App.tsx`:

- `/` - Главная страница
- `/sales` - Продажи
- `/purchases` - Закупки
- `/tasks` - Задачи и проекты
- `/resources` - Ресурсы
- `/finance` - Финансы
- `/logistics` - Логистика
- `/innovation` - Инноватика
- `/production` - Производство
- `/personnel` - Персонал
- `/company` - Компания
- `/marketing` - Маркетинг
- `/documents` - Документы
- `/processes` - Процессы
- `/settings` - Настройки

## Компоненты

### Header
Шапка приложения с:
- Отображением текущего времени
- Поисковой строкой
- Уведомлениями
- Меню пользователя

### Sidebar
Боковое меню навигации с:
- Список всех разделов
- Подсветкой активного раздела
- Адаптивным дизайном

### Layout
Основной layout компонент, который:
- Объединяет Header и Sidebar
- Предоставляет область для контента через `<Outlet />`
- Управляет состоянием sidebar

## Типы

Все типы определены в `src/types/index.ts`:
- `User` - данные пользователя
- `Notification` - уведомления
- `NavItem` - элементы навигации
- `SearchResult` - результаты поиска

## Хуки

### useCurrentTime
Возвращает текущее время в формате `HH:MM`, обновляется каждую секунду.

### useClickOutside
Закрывает элемент при клике вне его области. Используется в UserMenu и Notifications.
